
Python toolbox to read and process Seabird_ cnv files.

.. _Seabird: http://www.seabird.com/

These text files are the standard output files of the Seabird CTD software.


Install
-------

Developer
_________

Install as a user

.. code:: bash
	  
   python setup.py develop --user

Uninstall as a user
   
.. code:: bash
	  
   pip uninstall pycnv
